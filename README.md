# 📱 Мессенджер (Аналог Telegram)

Современный веб-мессенджер с реальным временем, разработанный на Node.js, React и WebSocket.

## ✨ Функционал

- 🔐 **Регистрация и авторизация**
- 💬 **Личные сообщения в реальном времени** (WebSocket)
- 📎 **Отправка файлов** (изображения, видео, аудио, документы)
- ✏️ **Редактирование сообщений**
- 🗑️ **Удаление сообщений**
- ↩️ **Ответы на сообщения**
- 📎 **Пересылка сообщений**
- 😊 **Реакции** (👍 ❤️ 😂 😮 😢 😡)
- 🟢 **Статус "онлайн/оффлайн"**
- ✍️ **Статус "печатает..."**
- ✓✓ **Галочки прочтения**
- 📱 **Адаптивный дизайн**

## 🛠️ Технологии

### Backend
- **Node.js** + **Express** — сервер
- **WebSocket** — реальное время
- **SQLite** — база данных
- **Multer** + **Sharp** — загрузка и обработка файлов
- **bcrypt** — хеширование паролей

### Frontend
- **React** — интерфейс
- **CSS** — стилизация
